# Changelog

## Unreleased

### Added

- Separate Galactic Command player view.
- Separate Galactic Director GM view.
- Server-side snapshot filtering.
- Fleet visibility levels: full, contact, and hidden.
- Director campaign-status module.
- GM command `convergence_director`.
- Foundation for fog of war and sensor intelligence.

### Changed

- Addon version increased to `0.8.0`.
- Player snapshots no longer expose hidden enemy fleets or GM-only data.
- Both interfaces use the same galaxy renderer and persistent simulation.

## 0.7.3

- Fixed Galaxy Renderer syntax and registration.
